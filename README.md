Gibbs samplers for posteriors from dynamic global-local shrinkage priors. To be used as a building block in Gibbs samplers for state-space models with global-local shrinkage process priors.

Currently only supports the dynamic shrinkage process prior in 

- Kowal et al. (2019). "Dynamic Shrinkage Processes", Journal of the Royal Statistical Society, Series B [arXiv](https://arxiv.org/abs/1707.00763)
